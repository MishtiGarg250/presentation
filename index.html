Slide 13 · Infrastructure

Backend Resilience & Scaling

  • Lazy Imports: Heavy ML imports (PyTorch, TF) deferred to first request — uvicorn binds within Render's 60s timeout
  • Thread-Safe Lock: Each service has threading.Lock() preventing double-loading
  • Sequential Warmup: Shadow → Sentiment → Prediction with gc.collect()
  • Redis Cache: Optional prediction cache (TTL 300s)
  • Dockerized: Full containerization for cloud deploy
  • Stripe Payments: Pro-tier via webhook integration
